# Library Management System (SQL Project)

## 🔹 Overview

This project is a relational database system for managing a library. It handles books, members, transactions, and availability tracking.

## 🔹 Features

* Book issue & return system
* Track available copies
* Member management
* Overdue book detection

## 🔹 Database Schema

* Authors
* Books
* Members
* Book Copies
* Transactions

## 🔹 Sample Queries

* Find available books
* Track borrowed books
* Identify overdue returns
* Most borrowed books

## 🔹 Tech Used

* MySQL
